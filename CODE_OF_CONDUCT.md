# Código de conducta

## Nuestro compromiso

Como participantes y responsables de este proyecto nos comprometemos a mantener
un entorno abierto, respetuoso y libre de acoso para toda persona, sin importar
edad, experiencia, identidad, origen o nivel técnico.

## Comportamiento esperado

- Usar lenguaje acogedor e inclusivo.
- Respetar puntos de vista y experiencias diferentes.
- Aceptar la crítica constructiva con apertura.
- Priorizar lo mejor para el proyecto y la comunidad.
- Mostrar empatia con las demás personas.

## Comportamiento inaceptable

- Lenguaje o imagenes sexualizadas y atención no deseada.
- Comentarios insultantes, despectivos y ataques personales o politicos.
- Acoso público o privado.
- Publicar información privada de terceros sin permiso.
- Compartir material que viole las reglas de seguridad del repositorio.

## Alcance

Este código aplica en todos los espacios del proyecto (issues, Pull Requests,
discusiones) y cuando alguien representa al proyecto en espacios públicos.

## Aplicación

Los casos de conducta abusiva pueden reportarse abriendo un issue con la
etiqueta correspondiente o contactando al responsable del repositorio. Todas las
quejas se revisaran y responderan de forma justa y proporcional.

Este código adapta el
[Contributor Covenant](https://www.contributor-covenant.org), versión 2.1.
