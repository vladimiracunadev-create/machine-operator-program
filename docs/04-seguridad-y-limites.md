# Seguridad y límites

## Uso previsto

Este repositorio es para documentación, aprendizaje general, historia y diseño de simuladores. No reemplaza entrenamiento real, certificaciones, licencias ni manuales oficiales vigentes.

## Límites

No incluir:

- procedimientos tacticos reales;
- instrucciones de uso de armas;
- instrucciones para evadir controles, autoridades o sistemas de seguridad;
- información clasificada, restringida o no pública;
- pasos que permitan operar maquinaria peligrosa sin supervisión;
- detalles técnicos sensibles sobre sistemas militares modernos.

## Enfoque permitido

Si una máquina tiene uso militar o de alto riesgo, documentar:

- historia pública;
- función general;
- mandos de simulación no sensibles;
- principios físicos;
- roles de tripulación a nivel general;
- reglas de seguridad;
- representación responsable en juegos.

## Recomendación editorial

Cuando exista duda, escribir para simulación educativa y no para operación real.
