# Recursos generales

Esta carpeta organiza material de apoyo transversal: esquemas, tablas, enlaces,
imagenes y glosarios específicos que ayudan a documentar y simular las máquinas.

## Que guardar aquí

- Diagramas y esquemas de mandos o sistemas (propios o de dominio permitido).
- Tablas de referencia (unidades, conversiones, rangos típicos).
- Enlaces públicos a museos, archivos, reglamentos y material académico.
- Glosarios específicos de un vehículo que amplian `docs/05-glosario-general.md`.

## Que no guardar

- Material con derechos que no permitan su redistribución.
- Información clasificada, restringida o sensible.
- Archivos binarios grandes sin curar (usar enlaces cuando sea posible).

## Formato recomendado

- Imagenes en formatos abiertos (`.svg`, `.png`) con nombre descriptivo.
- Un archivo `enlaces.md` por vehículo con la procedencia de cada recurso.
- Toda fuente externa registrada también en `manuales/fuentes.md`.

## Registro mínimo por recurso

| Campo | Descripción |
| --- | --- |
| Nombre | Nombre del archivo o del enlace. |
| Tipo | Diagrama, tabla, imagen, enlace o glosario. |
| Origen | Autor, institución o URL pública. |
| Licencia | Condición de uso o dominio. |
| Vehículo | Vehículo o sección relacionada. |
| Uso previsto | Para que sirve en la documentación o simulación. |
