# Recursos generales

Esta carpeta organiza material de apoyo transversal: esquemas, tablas, enlaces,
imagenes y glosarios especificos que ayudan a documentar y simular las maquinas.

## Que guardar aqui

- Diagramas y esquemas de mandos o sistemas (propios o de dominio permitido).
- Tablas de referencia (unidades, conversiones, rangos tipicos).
- Enlaces publicos a museos, archivos, reglamentos y material academico.
- Glosarios especificos de un vehiculo que amplian `docs/05-glosario-general.md`.

## Que no guardar

- Material con derechos que no permitan su redistribucion.
- Informacion clasificada, restringida o sensible.
- Archivos binarios grandes sin curar (usar enlaces cuando sea posible).

## Formato recomendado

- Imagenes en formatos abiertos (`.svg`, `.png`) con nombre descriptivo.
- Un archivo `enlaces.md` por vehiculo con la procedencia de cada recurso.
- Toda fuente externa registrada tambien en `manuales/fuentes.md`.

## Registro minimo por recurso

| Campo | Descripcion |
| --- | --- |
| Nombre | Nombre del archivo o del enlace. |
| Tipo | Diagrama, tabla, imagen, enlace o glosario. |
| Origen | Autor, institucion o URL publica. |
| Licencia | Condicion de uso o dominio. |
| Vehiculo | Vehiculo o seccion relacionada. |
| Uso previsto | Para que sirve en la documentacion o simulacion. |
